## Usage

1. **To login as a Doctor:**
   - Select "Doctor" role on login screen
   - Enter any doctor email and password from above
   - You'll see all patients in the system
   - Click on a patient to view details and assign them to yourself

2. **To login as a Patient:**
   - Select "Patient" role on login screen
   - Enter any patient email and password from above
   - You'll see your dashboard with exercises and health data

## Notes

- All preset users are created automatically on app startup
- If a user already exists, it won't be recreated (safe to restart app)
- Patients start unassigned - doctors can assign them from the patient detail screen
- You can still create new users via the Signup screen
